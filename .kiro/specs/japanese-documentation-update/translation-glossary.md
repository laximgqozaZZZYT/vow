## 文章表現パターン

### 説明文
- "VOW helps you..." → "VOWは...をサポートします"
- "Get started quickly..." → "すぐに始められます"
- "Track your progress..." → "進捗を管理できます"

### 機能説明
- "Daily habit tracking with..." → "...による日々の習慣管理"
- "Simple goal planning..." → "シンプルな目標計画"
- "Lightweight, privacy-first..." → "軽量でプライバシー重視の..."

### 手順説明
- "Clone the repository" → "リポジトリをクローンする"
- "Install dependencies" → "依存関係をインストールする"
- "Start development server" → "開発サーバーを起動する"

## 注意事項

1. **一貫性**: 同じ英語用語は常に同じ日本語に翻訳する
2. **自然さ**: 日本語として自然な表現を優先する
3. **読みやすさ**: 長すぎる文章は適切に分割する
4. **専門用語**: 開発者向けの内容では適切な技術用語を使用する
5. **ユーザー向け**: 一般ユーザー向けの内容では分かりやすい表現を使用する
