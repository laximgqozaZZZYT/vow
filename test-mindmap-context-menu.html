Mindmap Context Menu Test

Test Instructions:

  1. Open the Mindmap widget in your dashboard
  2. Right-click on any node to open the context menu
  3. Verify the menu appears with the following options:
    • Edit Text - White text with pencil icon
    • Register as Habit - Green text with cycle icon
    • Register as Goal - Green text with target icon
    • Delete Node - Red text with trash icon
  4. Test each menu option to ensure they work correctly
  5. Verify that habit nodes get green borders and goal nodes get purple borders

Context Menu Features Implemented:

✅ Right-click context menu handler (onNodeContextMenu)
✅ Menu positioning (top/left/right/bottom based on screen position)
✅ Menu styling with proper colors:
  • Edit Text: White text
  • Register as Habit: Green text
  • Register as Goal: Green text
  • Delete Node: Red text
✅ Node type system with visual feedback:
  • Default nodes: Gray border
  • Habit nodes: Green border
  • Goal nodes: Purple border
✅ Menu functionality:
  • Edit Text: Starts text editing mode
  • Register as Habit: Changes node type and calls onRegisterAsHabit
  • Register as Goal: Changes node type and calls onRegisterAsGoal
  • Delete Node: Removes node and connected edges

Technical Implementation Details:

Context Menu Handler:
- Uses onNodeContextMenu in React Flow
- Prevents default browser context menu
- Calculates menu position based on click location
- Handles screen edge detection for menu placement
Node Type System:
- Extended CustomNodeData interface with nodeType field
- Dynamic styling based on node type
- Visual feedback with border colors
- State management for node type changes
Custom Node Integration:
- Added onContextMenu handler to custom nodes
- Prevents event propagation conflicts
- Maintains compatibility with other node interactions

Expected Behavior:

  1. Right-click on node → Context menu appears
  2. Click "Edit Text" → Node enters edit mode with text input
  3. Click "Register as Habit" → Node border turns green, habit registration callback called
  4. Click "Register as Goal" → Node border turns purple, goal registration callback called
  5. Click "Delete Node" → Node and its connections are removed
  6. Click outside menu → Menu closes
  7. Press Escape → Menu closes

Troubleshooting:

If the context menu doesn't appear:
  • Check browser console for JavaScript errors
  • Verify that onNodeContextMenu is being called
  • Ensure the custom node's onContextMenu handler isn't preventing the event
  • Check that the menu positioning calculations are correct
