# スクリーンショット

このディレクトリにはREADME.md用のスクリーンショットを配置します。

## 必要な画像

### demo-preview.png
- サイズ: 1600x900px 推奨
- 内容: ランディングページのデモセクションのスクリーンショット
- 取得方法: https://main.do1k9oyyorn24.amplifyapp.com/ にアクセスし、デモセクションをスクリーンショット

### 画像の追加方法

1. ブラウザでサイトにアクセス
2. デモセクションまでスクロール
3. スクリーンショットを撮影
4. `demo-preview.png` として保存
5. このディレクトリに配置
6. git add & commit & push
