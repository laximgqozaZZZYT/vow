# Slack連携 ユーザー向けセットアップガイド

VOWアプリでSlack連携を有効にすると、以下の機能が利用できます：

- 📝 Slackから習慣の完了を報告
- ⏰ 習慣のリマインダー通知
- 📊 週次レポートの受信

## 連携手順

### 1. 設定ページにアクセス

1. VOWアプリにログイン
2. 右上のメニューから「Settings」をクリック
3. 「Integrations」タブを選択

### 2. Slackを連携

1. 「Connect Slack」ボタンをクリック
2. Slackの認証画面が表示されます
3. 連携するワークスペースを選択
4. 「許可する」をクリック
5. VOWの設定ページに戻り、「Connected」と表示されれば完了

### 3. 通知設定

連携後、以下の設定が可能です：

- **Habit reminders via Slack**: 習慣のリマインダーをSlackで受信
- **Weekly reports via Slack**: 週次レポートをSlackで受信
  - 曜日と時間を選択可能

### 4. 接続テスト

「Test Connection」ボタンをクリックすると、SlackのDMにテストメッセージが送信されます。

## Slackコマンド

連携後、Slackで以下のコマンドが使用できます：

| コマンド | 説明 |
|---------|------|
| `/habit-done [習慣名]` | 習慣の完了を報告 |
| `/habit-list` | 今日の習慣一覧を表示 |
| `/habit-status` | 習慣の進捗状況を表示 |

### 使用例

```
/habit-done 朝の運動
/habit-list
/habit-status
```

## トラブルシューティング

### 「Connect Slack」ボタンが反応しない

- ブラウザのポップアップブロッカーを確認
- ページをリロードして再試行

### 連携後にエラーが表示される

- 一度「Disconnect」してから再連携
- ブラウザのキャッシュをクリア

### Slackコマンドが動作しない

- VOWアプリにログインしているか確認
- Slack連携が有効か設定ページで確認
- ワークスペースにVOWアプリがインストールされているか確認

## 連携の解除

1. 設定ページの「Integrations」タブを開く
2. 「Disconnect」ボタンをクリック
3. 確認ダイアログで「OK」をクリック

連携を解除すると、Slackからの通知は停止しますが、VOWアプリ内のデータは保持されます。
